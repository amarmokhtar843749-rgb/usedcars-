
# Used Cars Market Analysis

## Project Overview

This project focuses on analyzing a used cars dataset to explore pricing trends and factors that influence vehicle prices in the used car market.

The goal of this project is to perform data cleaning, exploratory data analysis, and visualization to extract useful insights about used vehicles.

## Dataset

The dataset used in this project was obtained from Kaggle. It contains information about used vehicles such as brand, model, year, mileage, fuel type, transmission, and price.

The dataset provides an opportunity to explore how different factors affect the price and popularity of used cars.

## Project Objectives

The main objectives of this project are:

* Understand the structure of the dataset
* Clean and preprocess the data
* Identify important factors that influence car prices
* Analyze trends in the used car market
* Visualize key insights using charts and dashboards

## Data Cleaning

The following data cleaning steps will be applied:

* Handling missing values
* Removing duplicate records
* Correcting inconsistent data
* Converting data types when needed

## Tools and Technologies

The following tools will be used in this project:

* SQL for data cleaning and querying
* Excel for initial exploration
* Power BI for visualization and dashboards

## Expected Insights

Through this analysis, we aim to discover:

* The relationship between mileage and price
* Which car brands have higher resale value
* The most common fuel types and transmissions
* Price trends across different car models and years

## Project Status

This project is currently in progress. Further analysis and visualizations will be added.
